#### 2. 修改数据加载器

```java
// 修改 parseTrade() 方法
static TradeEv parseTrade(String line) {
    String[] parts = line.split(",");
    
    // 原有字段
    long ts = Long.parseLong(parts[0]);
    int accountId = Integer.parseInt(parts[1]);
    int symbolId = Integer.parseInt(parts[2]);
    byte side = parts[3].equals("B") ? (byte)1 : (byte)-1;
    double execPrice = Double.parseDouble(parts[4]);
    
    // 新增字段（带容错）
    String eventText = parts.length > 5 ? parts[5] : null;
    double orderSize = parts.length > 6 ? Double.parseDouble(parts[6]) : 0.0;
    Double tp = (parts.length > 7 && !parts[7].isEmpty()) 
        ? Double.parseDouble(parts[7]) : null;
    Double sl = (parts.length > 8 && !parts[8].isEmpty()) 
        ? Double.parseDouble(parts[8]) : null;
    
    return new TradeEv(ts, accountId, symbolId, side, execPrice, 
                      eventText, orderSize, tp, sl);
}
```

#### 4. 添加 Symbol 映射（可选但推荐）

```java
final class SymbolMapper {
    private final Map<Integer, String> idToName = new HashMap<>();
    
    void register(int id, String name) {
        idToName.put(id, name);
    }
    
    String getName(int id) {
        return idToName.getOrDefault(id, "UNKNOWN");
    }
    
    static SymbolMapper createDefault() {
        SymbolMapper m = new SymbolMapper();
        m.register(1, "EURUSD");
        m.register(2, "GBPUSD");
        m.register(3, "USDJPY");
        // ... 其他货币对
        return m;
    }
}
```
