### 核心检测能力

#### 1. 机器人特征（Bot Signature）
- ✅ CV < 0.15 → 时间间隔极度规律
- ✅ 固定止盈止损模式（identical_tpsl_ratio > 0.8）
- ✅ 低订单大小熵（entropy < 1.0）
- ✅ 超快订单占比高（pct_lt_300ms > 0.5）

#### 2. 延迟套利（Stale Quote Arbitrage）
- ✅ Quote Age > 200ms 且 Markout 为正
- ✅ 请求价格偏差持续异常
- ✅ 负时间差比例高（客户端时间超前）

#### 3. 批量操控（Batch Manipulation）
- ✅ 多账户500ms内同向交易
- ✅ 账户间余弦相似度 > 0.9
- ✅ 相同IP段大量账户

#### 4. 账户共享（Account Sharing）
- ✅ 单账户多个登录名
- ✅ 频繁切换客户端类型/版本
- ✅ IP切换频繁

## 技术架构

### 新增算法实现

**Shannon Entropy** (OrderSizeAnalyzer)
```
H = -Σ p(i) * log2(p(i))
其中 p(i) = count(size_i) / total_count
```

**Coefficient of Variation** (IntervalStats)
```
CV = std / mean
CV < 0.15 → 高度可疑（机器人）
```

**Z-Score Normalization** (RiskConfig)
```
z = (x - μ) / σ
综合评分 = Σ (w_i * z_i)
```

**Cosine Similarity** (SyncDetector)
```
similarity = dot(A, B) / (||A|| * ||B||)
相似度 > 0.9 → 关联账户
```

### 数据结构

**CircularBuffer** (AccountState)
```java
long[] recentOrderTimes = new long[100];  // 最近100笔订单时间
int orderTimeIndex = 0;                   // 循环指针
```

**TimeBucketing** (SyncDetector)
```java
Map<Long, Map<Key, List<Long>>> buckets;
// 500ms时间桶 → {symbol+side} → 账户列表
```
